# floor()

Функция **`floor`** осуществляет округление численного аргумента до ближайшего не большего целого.

## Синтаксис

```
number floor( number )
```

## Описание и примеры

```
floor(2.3) ? 2
floor(-2.3) ? -3
floor(-1 div 0) ? -Infinity
floor('zero') ? NaN
```
